# withcapsule

This organization hosts the source code for [Capsule](https://withcapsule.dev), a temporary file-transfer service.

### Repositories

| Repo | Description |
|---|---|
| [Server](https://github.com/withcapsule/Server) | Rust / Axum HTTP backend |
| [CLI](https://github.com/withcapsule/CLI) | Terminal client with encryption support |
| [Android](https://github.com/withcapsule/Android) | Kotlin / Compose Android app |
| [WebUI](https://github.com/withcapsule/WebUI) | Static web client + Cloudflare Worker |
| [Docs](https://github.com/withcapsule/Docs) | Documentation site (Astro / Starlight) |
