## Capsule - Simple File Transfer

No accounts. No setup. Upload a file, get a link and QR code back. Files delete themselves after an hour.

<!--![Capsule demo](../demo/video2.gif)-->

**Use it right now at [withcapsule.dev](https://withcapsule.dev?utm_source=github&utm_medium=readme&utm_campaign=header) or via curl:**
```sh
curl -F "f=@file.txt" https://send.withcapsule.dev/upload
```
